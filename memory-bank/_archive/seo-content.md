## Page Template (for `/for/` pages)

```
1. Hero
   - Pain point headline specific to tool
   - Subhead: "Ship fast with [Tool]. Did that change work? Loupe tells you."
   - CTA: Free audit

2. Problem Section
   - What breaks when shipping fast with [tool]
   - Specific examples (headlines change, CTAs disappear, etc.)

3. Solution Section
   - How Loupe catches it
   - Screenshot → LLM analysis → correlation

4. Social Proof
   - Testimonials (when available)
   - "Join X founders monitoring their sites"

5. CTA
   - Free audit (same as homepage)
```
